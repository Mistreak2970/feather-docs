# Action tags

| Tag       | Condition / Meaning                             |
| --------- | ----------------------------------------------- |
| jumping   | Player is currently jumping                     |
| climbing  | Player is climbing a ladder/vine/etc.           |
| emoting   | Player is performing an emote                   |
| falling   | Player is falling (in the air, moving downward) |
| flying    | Player is flying (creative/elytra/other flight) |
| gliding   | Player is gliding (e.g. with elytra)            |
| inwater   | Player is in water                              |
| onground  | Player is on the ground (standing on a block)   |
| sleeping  | Player is sleeping in a bed                     |
| sneaking  | Player is sneaking/crouching                    |
| sprinting | Player is sprinting                             |
| swimming  | Player is swimming                              |

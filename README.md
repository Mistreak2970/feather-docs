# Welcome

Welcome to the docs for Feather Essentials - the colorful and simple Minecraft Bedrock Edition add-on for server management!

### Getting Started with Feather Essentials

Feather Essentials is the best Add-on to make creating your server easier.

1. **Install**: Download and install the add-on from CurseForge
2. **Import: Import the Add-on into Minecraft by opening it with Minecraft**
3. Activate: Activate the Add-On by turning on Beta APIs and activating both the Resource Pack and Behavior Pack.

# Basics

{% hint style="info" %}
To navigate to the UI Builder, go to Config UI and click "UI Builder"
{% endhint %}

Feather Essentials is built around the UI Builder, a semi-advanced UI creator.

{% hint style="info" %}
It is highly recommended to mess around with the UI Builder instead of reading this.
{% endhint %}

## What is the UI Builder?

The UI Builder is a feature in the addon that has seemingly endless possibilities. It's currently in beta.

You can make loads of things including:

Custom admin panel

Server menu

Warps menu

..and more!
